# 个人网站

这是一个可直接打开的静态个人网站。你可以在网页里填写个人信息、经历、项目和联系方式，也可以导出一份已经写入资料的 `index.html` 用于发布。

## 本地使用

1. 双击打开 `index.html`。
2. 点击右上角编辑按钮。
3. 填写个人资料后点击“保存预览”。
4. 点击“导出网页”可以下载一份包含当前资料的 `index.html`。

## 让别人通过网址访问

把这个文件夹里的 `index.html`、`styles.css`、`script.js` 上传到任意静态网站托管服务即可，例如：

- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages

如果你已经有域名，可以在托管服务里绑定自己的域名；如果没有，托管服务通常也会提供一个免费访问网址。

## 可编辑内容

- 姓名、头像、个人简介
- 多张个人图片，页面会按时间自动切换
- 当前身份、城市、邮箱、电话
- 经历列表
- 项目或作品列表
- 社交主页与简历链接

## 图片网址怎么获得

网站里的图片需要填写可以公开访问的图片网址，通常以 `https://` 开头，并且最好以 `.jpg`、`.png`、`.webp` 结尾。

简单做法：

1. 把图片上传到你的网站托管平台、图床、对象存储，或公开相册。
2. 打开图片，复制“图片地址”。
3. 在编辑面板的“个人图片网址，每行一个”里粘贴图片地址。
4. 多张图片就每行放一个网址。

如果你准备把网站部署到 GitHub Pages，也可以把图片放进项目里的 `images` 文件夹，再填写类似 `images/me-1.jpg` 的相对路径。
